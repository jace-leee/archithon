# Elice Cloud (엘리스 클라우드) 리서치

## 1. 회사 및 제품 개요

**엘리스(Elice)**는 2015년에 설립된 한국의 AI 교육 및 클라우드 솔루션 기업으로, 서울에 본사를 두고 있습니다. 처음에는 AI·코딩 교육 플랫폼으로 시작했으나, 이후 AI 인프라 및 클라우드 사업으로 영역을 확장하였습니다. 현재는 "AI 시대의 기반(Foundation of the AI Era)"을 슬로건으로 내세우며 교육·인프라·플랫폼·콘텐츠 전 영역의 AI 솔루션 기업으로 포지셔닝하고 있습니다.

**엘리스클라우드(Elice Cloud)**는 엘리스그룹의 AI 특화 클라우드 서비스 브랜드로, 고성능 GPU 및 국산 NPU를 기반으로 AI 연구·개발·배포에 필요한 All-in-One 클라우드 환경을 제공합니다.

## 2. 핵심 서비스 및 기능

### ① Elice Project
- 학생·연구자·개발자를 위한 월정액 구독형 서비스
- 사전 구성된 개발 환경과 GPU 자원을 온라인 플랫폼에서 즉시 사용
- JupyterLab, Visual Studio Code, SSH 등 개발 도구 내장 지원

### ② Elice Cloud On-Demand
- 기업 및 연구기관 대상, 원하는 사양의 GPU 컨테이너 환경을 탄력적으로 할당
- On-Demand(즉시 사용)와 Reserved(기간 예약) 방식 동시 제공
- 독자 개발한 **동적 할당 기술(Dynamic Allocation Technology)**로 자원 낭비 최소화

### ③ Elice Cloud ML API
- 별도 설치·설정 없이 오픈소스 AI 모델(DeepSeek R1, Meta Llama 등)을 API 형태로 즉시 활용
- 맞춤형 GPU·NPU 환경에서 실행되는 프라이빗 클라우드 구조로 데이터 보안 확보
- 2025년 2월 기준 ML API 라이브러리 기능 업그레이드 발표

### ④ Elice Cloud DataHub
- AI 모델 연구·개발에 최적화된 객체 스토리지 기반 제품
- 대용량 비정형 데이터(이미지, 텍스트, 영상 등)를 클라우드 환경에서 생성·저장·분석

## 3. 지원 하드웨어 (GPU/NPU)

- **NVIDIA GPU**: A100 (40GB/80GB), H100 (InfiniBand 클러스터 기반), H200
- **국산 NPU**: 퓨리오사AI(FuriosaAI), 리벨리온(Rebellions)의 최신 NPU 지원
  - 국내 클라우드 서비스 중 최초로 국산 NPU를 정식 서비스에 편입

## 4. 가격 모델

| GPU 모델 | VRAM | 시간당 요금 |
|---|---|---|
| NVIDIA A100 | 80GB | 약 2,000원/시간 |
| NVIDIA A100 | 40GB | 약 1,380원/시간 |

- 글로벌 클라우드(AWS, Azure, GCP)와 동일 조건 대비 **77% 이상 저렴**
- 국내 인프라 자체 구축으로 해외 클라우드 대비 원가 절감 실현
- On-Demand(초단기 사용) 및 Reserved(기간 예약) 요금제 병행 운영
- **중소기업 클라우드 바우처 지원사업** 연계로 정부 보조금 활용 가능(2025년 기준)

## 5. 타겟 사용자 및 시장

**주요 고객군:**
- 대학교 및 학술 연구기관 (교육부 산하 연구기관 포함)
- AI 스타트업 및 중소기업
- 대기업 AI 개발팀
- 공공기관 (CSAP IaaS 인증으로 공공 클라우드 시장 공략)
- AI 교육 수강생 및 개인 개발자

**규모:** 2025년 5월 기준 **4,800개 이상 고객 기관** 확보. 2023년 6월~2025년 3월(21개월) 사이 사용자 수 **약 74배 증가**, 2024년 11월~2025년 5월(6개월) 사이 최신 GPU/NPU 이용 기관 수 **9배 급증**.

## 6. 타 클라우드 플랫폼과의 차별점

| 항목 | 엘리스클라우드 | AWS/Azure/GCP |
|---|---|---|
| 가격 | 동일 조건 대비 77% 이상 저렴 | 고가 |
| 국산 NPU 지원 | 퓨리오사AI, 리벨리온 NPU 공식 지원 | 미지원 |
| 데이터 주권 | 국내 서버만 사용, 외부 유출 없음 | 해외 서버 포함 |
| 보안 인증 | 국내 CSAP IaaS 인증 취득 | 국내 공공 클라우드 요건 미충족 경우 있음 |
| 동적 할당 | 자체 기술로 자원 낭비 최소화 | 표준 컨테이너 할당 |
| AI 교육 연계 | 교육 LMS 플랫폼과 통합 가능 | 별도 구축 필요 |
| 모듈러 데이터센터 | PMDC(이동형 모듈러 데이터센터) 구축 중 | 해당 없음 |

## 7. 최근 주요 뉴스 및 동향 (2025~2026)

### Series C 투자 유치 (2025년 9월)
글로벌 투자자로부터 200억원(약 1,490만 달러) 규모의 시리즈 C 투자를 유치하며 APAC 확장에 속도를 내고 있습니다.

### AI PMDC(이동형 모듈러 데이터센터) 구축 (2025년)
NVIDIA H100 5,000장 수용 규모의 1MW급 AI PMDC를 설계·구축 중. 국내 클라우드 기업 최대 규모 예상. PUE(전력 효율 지수) 1.27 달성. 모듈러 데이터센터로는 국내 최초 **CSAP IaaS 인증** 취득.

### 델(Dell)과의 파트너십 (2025 DTW)
Dell Technologies World 2025에서 Dell이 한국 파트너 대표 사례로 엘리스를 소개. Dell 하드웨어 인프라에 엘리스의 4년간 자체 개발 소프트웨어 클라우드를 결합한 GPU 인프라 국산화 추진.

### 네이버클라우드와 GPU/NPU 팜 구축 협약
네이버클라우드와 협력하여 클라우드 기반 GPU·NPU 팜을 공동 구축, 국내 AI 인프라 생태계 확장에 기여.

### 과기정통부 장관 표창 (2025)
클라우드 산업 발전 공로로 과학기술정보통신부 장관 표창 수상.
